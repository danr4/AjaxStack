An online demo of the HTML5 JSON Report format to view arbitrary JSON data in a human-readable report format.

Usage:

Paste some JSON text in the TEXTAREA below and click View JSON Report button.

Download for use in your own projects

jsonreport.js is a stand-alone javascript library you can embed in your own pages by adding a reference to the js library: 

<script type="text/javascript" src="jsonreport.js"></script>

this adds a global object _.jsonreport(json) you can call to turn your JSON into human-readable HTML. e.g: 

var html = _.jsonreport('{"any":"json"}');

Note: to use the default jsonreport styles, add the .jsonreport className to the containing element, e.g: 

<div class="jsonreport"><div>

Complete example: 

document.getElementsByClassName('jsonreport')[0].innerHTML = _.jsonreport('{"any":"json"}');
